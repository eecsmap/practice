Independency

If we have:

\[P(A) \cdot P(B) = P(A,B)\]
\[P(A) = P(A|B)\]

We say that \(A\) is independent with \(B\), as \(A \perp B\)

import random

def sample():
    a_done = False
    count = 0
    while not a_done:
        a = random.random() < 0.5
        count += 1
        #print(a, b)
        if count == 1:
            old_a = a
        else:
            if a != old_a: a_done = True
    return count

n = 100000
print(sum(sample() for i in range(n)) / n)
