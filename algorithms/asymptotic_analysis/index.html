Basic facts

  • \(logn \in o(n^\epsilon)\) for any \(\epsilon > 0\)
  • \(log^{c}n \in o(n^\epsilon)\) for any \(\epsilon > 0\)
  • \(c_1^{n} \in o(c_2^{n})\) for all \(1 \leq c_1 < c_2\)
